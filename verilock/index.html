How It Works

Every transaction goes through this flow

1

Request Approval

Agent sends transaction details to the Notary API: destination, amount, purpose. Notary checks against YAML policy.

2

Notary Issues Token

If policy allows, the Notary issues a signed EIP-712 approval token. The signature is cryptographically bound to this specific transaction.

3

Encode & Submit

The agent encodes the token into the transaction data using the SDK, then submits to the Safe for execution.

4

Guard Validates

Before the Safe executes, the Guard checks the token: valid signature? Not expired? Destination matches? Amount within limit? All checks pass → execute.

// Try sending ETH without approval token
Safe.sendTransaction({
  to: "0x090f...ED563",
  value: "0.02 ETH",
  data: "0x" // No token → Guard rejects
})

// Result: REVERTED by Guard
// Safe UI: "Simulation Error: TokenMissing()"
// 1. Get approval from Notary
const token = await notary.approve({
  agent: "trading-bot-01",
  destination: "0x090f...ED563",
  amount_usd: 20,
  purpose: "yield_deposit"
});

// 2. Encode and submit
const data = VerilockSDK.encode(calldata, token);
await Safe.sendTransaction({ to, value, data });

// Result: SUCCESS
function checkTransaction(..., bytes data) external {
  // Only Safe can call
  require(msg.sender == safeAddress);

  // Extract and validate token
  ApprovalToken memory token = _extractToken(data);
  require(_verifySignature(token), "Invalid signature");
  require(!_isExpired(token), "Token expired");

  // All checks pass → allow
}

Key Features

What makes Verilock secure

🛡️

Immutable Guard

Deployed on-chain, the Guard enforces rules at the blockchain level. No one can bypass it.

📋

Hot-Reloadable Policy

YAML policy updates in real-time without redeploying the smart contract.

🔐

EIP-712 Signatures

Cryptographic proof that the Notary approved this specific transaction.

🔥

Escape Hatch

Safe owners can always remove the Guard via setGuard(address(0)). No lock-in.

📊

Three-Tier Approval

Auto-Tier 1, Notify + Veto Tier 2, or Human-in-the-loop Tier 3 based on amount and history.

🎭

Replay Protection

Each token has a unique nonce consumed after use. Never reusable.
